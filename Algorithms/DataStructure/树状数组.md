# 树状数组

## 代码

时间复杂度 $O(\log n)$（单次操作）。

```cpp
int lowbit(int x) {
    return x & -x;
}
struct Fenwick {
    ll c[MAXN];
    ll query(int x) {
        ll ret = 0;
        for (int i = x; i; i -= lowbit(i))
            ret += c[i];
        return ret;
    }
    void add(int x, ll k) {
        for (int i = x; i <= n; i += lowbit(i))
            c[i] += k;
    }
};
```

## 注

这份代码支持 **单点加** 和 **前缀和**。
