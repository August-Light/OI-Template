# 并查集

## 代码

时间复杂度：
- 初始化 $\Theta(n)$。
- 单次操作均摊 $O(\alpha(n))$。

空间复杂度 $\Theta(n)$。

```cpp
struct DSU {
    int fa[MAXN];
    void init(int n) {
        for (int u = 1; u <= n; u++)
            fa[u] = u;
    }
    int find(int u) {
        return fa[u] == u ? u : (fa[u] = find(fa[u]));
    }
    void merge(int u, int v) {
        fa[find(v)] = find(u);
    }
    bool same(int u, int v) {
        return find(u) == find(v);
    }
};
```

## 注

代码为只应用路径压缩的版本。**时间复杂度有所不同**。

同时使用路径压缩与启发式合并的代码如下：

```cpp
struct DSU {
    int fa[MAXN], siz[MAXN];
    void init(int n) {
        rep(u, 1, n)
            fa[u] = u, siz[u] = 1;
    }
    int find(int u) {
        return u == fa[u] ? u : (fa[u] = find(fa[u]));
    }
    void merge(int u, int v) {
        u = find(u), v = find(v);
        if (u == v)
            return;
        if (siz[u] > siz[v])
            swap(u, v);
        fa[u] = v;
        siz[v] += siz[u];
    }
    bool same(int u, int v) {
        u = find(u), v = find(v);
        return u == v;
    }
};
```
