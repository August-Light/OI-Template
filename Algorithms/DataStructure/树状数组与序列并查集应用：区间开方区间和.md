# 区间开方区间和

## 代码

时间复杂度：（$n$ 与 $m$ 同阶）

- 预处理 $\Theta(n)$。
- 单次修改 $O(\log \log W \log n)$。其中 $O(\log \log W)$ 是开方开到 $1$ 的次数。
- 单次查询 $O(\log n)$。

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

const int MAXN = 1e5 + 5;

int n, m;
ll a[MAXN];

// 省略树状数组

int fa[MAXN];
void init(int n) {
    for (int u = 1; u <= n; u++)
        fa[u] = u;
}
int find(int u) {
    if (fa[u] == u)
        return u;
    return fa[u] = find(fa[u]);
}

Fenwick tr;

int main() { ios::sync_with_stdio(0); cin.tie(0);
    cin >> n;
    init(n+1);
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        tr.add(i, a[i]);
    }
    cin >> m;
    while (m--) {
        int op; cin >> op;
        int l, r; cin >> l >> r; if (l > r) swap(l, r);
        if (op == 0) {
            int i = find(l);
            while (i <= r) {
                ll t = sqrt(a[i]);
                tr.add(i, t - a[i]);
                a[i] = t;
                if (a[i] == 1)
                    i = fa[i] = find(i+1);
                else
                    i++;
            }
        } else if (op == 1)
            cout << tr.query(r) - tr.query(l-1) << '\n';
    }
    return 0;
}
```
