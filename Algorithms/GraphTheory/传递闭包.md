# Floyd

## 代码

时间复杂度 $\Theta\left(\dfrac {n^3} w\right)$。

空间复杂度 $\Theta\left(\dfrac {n^2} w\right)$。

```cpp
bitset<MAXN> dis[MAXN];
void Floyd() {
    for (int k = 1; k <= n; k++)
        for (int i = 1; i <= n; i++)
            if (dis[i][k])
                dis[i] |= dis[k];
}
```
