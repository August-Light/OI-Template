# 摩尔投票

## 代码

时间复杂度 $\Theta(n)$。

空间复杂度 $\Theta(1)$。

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int n;

int main() { ios::sync_with_stdio(0); cin.tie(0);
    cin >> n;
    int ans = 0, cnt = 0;
    for (int i = 1; i <= n; i++) {
        int a; cin >> a;
        if (ans == a)
            cnt++;
        if (cnt == 0)
            ans = a, cnt++;
        if (ans != a)
            cnt--;
    }
    cout << ans << '\n';
    return 0;
}

```

## 注

因为有空间复杂度限制，所以放完整代码。
