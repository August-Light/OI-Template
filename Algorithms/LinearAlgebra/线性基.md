# 线性基

## 代码

时间复杂度 $O(\log W)$（单次操作）。

```cpp
struct Basis {
    ull a[MAXT];
    void insert(ull x) {
        for (int i = T; i >= 0; i--)
            if (x >> i & 1) {
                if (!a[i]) {
                    a[i] = x;
                    return;
                }
                x ^= a[i];
            }
    }
    ull query_max(ull x = 0) {
        for (int i = T; i >= 0; i--)
            if ((x ^ a[i]) > x)
                x ^= a[i];
        return x;
    }
};
```

## 注

`query_max` 可求 $x$ 与线性基的最大异或和。
