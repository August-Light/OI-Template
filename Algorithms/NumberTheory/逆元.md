# 逆元

## 代码

时间复杂度 $O(\log n)$。

```cpp
// exgcd
ll inv(ll a) {
    ll x, y; exgcd(a, MOD, x, y);
    return (x % MOD + MOD) % MOD;
}

// Fermat
ll inv(ll a) {
    return qpow(a, MOD-2);
}
```
